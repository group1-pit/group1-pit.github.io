Phase 4: Implementation


We envision a few future developments for the app if we had more time:

  1. Library Patron Interviews - We would love to talk to patrons who already use the bus service and get their thoughts on an application like this.
  2. User Testing - Ideally, we would have liked to test this on actual library patrons and understand if the pain points we are trying to ease are actually being addressed within the design of our app.
  3. Broadening this App to More Services - Unemployment is just one of the services an app like this could serve. Eventually we would like this app to be applicable to more needs - like citizenship applications, GED testing, job applications, etc.
